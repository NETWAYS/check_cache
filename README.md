check_cache
===========

A Icinga/Nagios Plugin wrapper with caching function.

### Usage

    check_cache -H <hostname>
            -C <command>
            -S <servicename>
            -p <period>
            [-m] <mode>
            [-e] <executor>
            [-u] <username>
            [-t] <tmpdir>
            [-c] <configfile>
            [-v] [<path to logfile>]
            [-h]
            [-V]
            [--print-sample-config]
